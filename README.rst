Introduction
============

``ftw.globalstatusmessage`` displays a global static plone-like statusmessage.
Configurable throught the plone control panel.

Currently 3 types are supported:

- info (green)
- warning (orange)
- error (red)



Compatibility
=============

Runs with `Plone <http://www.plone.org/>`_ `4.1`, `4.2`, `4.3`.
